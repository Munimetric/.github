## Munimetric

Structured stress monitoring for community drinking-water systems across all 50 U.S. states.

Munimetric integrates public-source data from federal and state agencies to produce deterministic stress scores, binary signals, and peer-relative context for 9,500+ service markets.

**What we track:**
- **Stress Scores** — 0–100, five families, deterministic and threshold-based
- **Signals** — Binary events with severity, lifecycle, and provenance
- **Peer Context** — Cohort-relative percentiles and benchmarks
- **Fiscal Overlay** — Parent-government fiscal pressure as a monitoring layer
